3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # PS: https://blog.csdn.net/TYRA9/article/details/131399987

    # Method One: REGEXP_LIKE
    SELECT
        *
    FROM    
        `Patients`
    WHERE
        REGEXP_LIKE(`conditions`, '^DIAB1|\\sDIAB1');


    # --------------------
    # Method Two: 模糊查询 (LIKE)
    SELECT
        *
    FROM
        `Patients`
    WHERE
        `conditions` LIKE 'DIAB1%' 
        OR `conditions` LIKE '% DIAB1%';
    ```
