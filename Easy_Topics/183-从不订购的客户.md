#### **No.183-从不订购的客户**:
1. **力扣链接**：https://leetcode.cn/problems/customers-who-never-order/

3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: NOT EXISTS (robust)
    SELECT
        `name` AS `Customers`
    FROM 
        `Customers` `C`
    WHERE NOT EXISTS (
        SELECT 1
        FROM `Orders` `O`
        WHERE `O`.`customerId` = `C`.`id`
    );

    # -------------------------
    # Method Two : exclude id (easy to come up with)
    SELECT
        `name` AS `Customers`
    FROM 
        `Customers`
    WHERE `id` NOT IN (
        SELECT
            `customerId`
        FROM
            `Orders`
    );

    # -------------------------
    # Method Three : LEFT JOIN (most efficient)
    SELECT
        `name` AS `Customers`
    FROM 
        `Customers` `c`
    LEFT JOIN
        `Orders` `o` ON `c`.`id` = `o`.`customerId`
    WHERE
        `customerId` IS NULL;
    ```
