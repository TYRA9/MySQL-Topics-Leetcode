#### **No.175-组合两个表**:
1. **力扣链接**：https://leetcode.cn/problems/combine-two-tables/description
2. **答案**（纯代码）：  
      
    ```sql
    # Write your MySQL query statement below
    # Method One: 使用左连接
    SELECT
        `firstName`,
        `lastName`,
        `city`,
        `state`
    FROM
        `Person`
    LEFT JOIN
        `Address` ON `Person`.`personId` = `Address`.`personId`;

    # --------------------
    # Method Two: 使用右连接
    SELECT
        `firstName`,
        `lastName`,
        `city`,
        `state`
    FROM
        `Address`
    RIGHT JOIN
        `Person` ON `Person`.`personId` = `Address`.`personId`;
    ```
