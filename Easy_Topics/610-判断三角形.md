#### **No.610-判断三角形**:
1. **力扣链接**：https://leetcode.cn/problems/triangle-judgement/
2. **题目说明**
    - 
3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: CASE
    SELECT
        `x`,
        `y`,
        `z`,
        CASE 
            WHEN `x` + `y` > `z` AND `y` + `z` > `x` AND `x` + `z` > `y` THEN 'Yes'
            ELSE 'No'
        END AS `triangle`
    FROM
        `triangle`;

    # --------------------
    # Method Two : IF
    SELECT
        `x`,
        `y`,
        `z`,
        IF((`x` + `y` > `z` AND `y` + `z` > `x` AND `x` + `z` > `y`), 'Yes', 'No') AS `triangle`
    FROM
        `triangle`;
    ```
