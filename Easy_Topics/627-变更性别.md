3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One : IF
    UPDATE
        `Salary`
    SET
        `sex` = IF(`sex` = 'f', 'm', 'f');

    # -------------------------
    # Method Two : CASE WHEN
    UPDATE
        `Salary`
    SET
        `sex` = (
            CASE 
                WHEN `sex` = 'f' THEN 'm'
                ELSE 'f'
            END
        );

    # -------------------------
    # Method Three : ASCII() + CHAR()
    -- PS_1: ASCII(character): This function returns the ASCII (or more accurately, the character code in the database's character set) value of the first character in the input string.
    -- PS_2: CHAR(number): This function converts a numerical character code back into its corresponding character.

    UPDATE
        `Salary`
    SET 
        `sex` = CHAR(ASCII('f') + ASCII('m') - ASCII(`sex`));
    ```
