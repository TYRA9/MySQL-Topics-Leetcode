3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    SELECT
        `teacher_id`,
        COUNT(DISTINCT `subject_id`) AS `cnt`
    FROM
        `Teacher`
    GROUP BY
        `teacher_id`
    ```
