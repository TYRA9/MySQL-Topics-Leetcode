3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    -- To match an exact number of characters in a row, 
    -- we can use Curly Braces {}.
    SELECT
        `product_id`,
        `product_name`,
        `description`
    FROM
        `Products`
    WHERE
        REGEXP_LIKE(`description`, '\\bSN[0-9]{4}-[0-9]{4}\\b', 'c')
    ORDER BY
        `product_id`;
    ```
