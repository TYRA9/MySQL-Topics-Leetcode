#### **No.182-查找重复的电子邮箱**:

3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: 隐式自连接（多表查询）
    SELECT
        DISTINCT `p1`.`email` AS `Email`
    FROM 
        `Person` `p1`, `Person` `p2`
    WHERE
        `p1`.`email` = `p2`.`email`
    AND
        `p1`.`id` != `p2`.`id`;
        
    # -------------------------
    # Method Two: 显式自连接（内连接）
    SELECT
        DISTINCT `p1`.`email` AS `Email`
    FROM 
        `Person` `p1`
    INNER JOIN
        `Person` `p2` ON `p1`.`email` = `p2`.`email`
    WHERE
        `p1`.`id` != `p2`.`id`;

    # -------------------------
    # Method Three: 分组查询
    SELECT
        `email` AS `Email`
    FROM
        `Person`
    GROUP BY
        `email`
    HAVING 
        COUNT(*) > 1;
    ```
