3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: INNER JOIN (内连接)
    SELECT 
        `U`.`name`,
        SUM(`T`.`amount`) AS `balance`
    FROM
        `Users` `U`
    INNER JOIN
        `Transactions` `T` USING(`account`)
    GROUP BY
        `T`.`account`
    HAVING
        `balance` > 10000;

    # -------------------------
    # Method Two : 多表查询
    SELECT
        `U`.`name`,
        SUM(`T`.`amount`) AS `balance`
    FROM
        `Users` AS `U`, `Transactions` AS `T`
    WHERE
        `U`.`account` = `T`.`account`
    GROUP BY
        `T`.`account`
    HAVING
        `balance` > 10000;
    ```
