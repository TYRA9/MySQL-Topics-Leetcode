#### **No.1741-查找每个员工花费的总时间**:

3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    SELECT
        `event_day` AS `day`,
        `emp_id`,
        SUM(`out_time` - `in_time`) AS `total_time`
    FROM
        `Employees`
    GROUP BY
        `event_day`, `emp_id`;
    ```
