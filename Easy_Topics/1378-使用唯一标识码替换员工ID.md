3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: LEFT JOIN
    SELECT
        `unique_id`,
        `name`
    FROM
        `Employees`
    LEFT JOIN 
        `EmployeeUNI` ON `Employees`.`id` = `EmployeeUNI`.`id`;

    # -------------------------
    # Method Two: RIGHT JOIN
    SELECT
        `unique_id`,
        `name`
    FROM
        `EmployeeUNI`
    RIGHT JOIN
        `Employees` ON `EmployeeUNI`.`id` = `Employees`.`id`;
    ```
