3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: Eat a piece of sesame
    SELECT 
        *
    FROM
        `Cinema`
    WHERE
        `id` % 2 = 1
    AND
        `description` != 'boring'
    ORDER BY
        `rating` DESC;

    # -------------------------
    # Method Two : Just other form 
    SELECT 
        *
    FROM
        `Cinema`
    WHERE
        MOD(`id`, 2) = 1
    AND
        `description` NOT LIKE 'boring'
    ORDER BY
        `rating` DESC;
    ```
