#### **No.183-从不订购的客户**:
1. **力扣链接**：https://leetcode.cn/problems/customers-who-never-order/

3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: exclude name
    SELECT
        `name` AS `Customers`
    FROM 
        `Customers`
    WHERE `name` NOT IN (
        SELECT
            `c`.`name`
        FROM
            `Customers` `c`, `Orders` `o`
        WHERE
            `o`.`customerId` = `c`.`id`
    );

    # -------------------------
    # Method Two : exclude id
    SELECT
        `name` AS `Customers`
    FROM 
        `Customers`
    WHERE `id` NOT IN (
        SELECT
            `customerId`
        FROM
            `Orders`
    );

    # -------------------------
    # Method Three : LEFT JOIN
    SELECT
        `name` AS `Customers`
    FROM 
        `Customers` `c`
    LEFT JOIN
        `Orders` `o` ON `c`.`id` = `o`.`customerId`
    WHERE
        `customerId` IS NULL;
    ```
