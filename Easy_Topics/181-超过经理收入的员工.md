#### **No.181-超过经理收入的员工**:
1. **力扣链接**：https://leetcode.cn/problems/employees-earning-more-than-their-managers/description/
2. **题目说明**

3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: 隐式自连接（多表查询）
    SELECT
        `emp1`.`name` AS `Employee`
    FROM
        `Employee` `emp1`, `Employee` `emp2`
    WHERE
        `emp1`.`managerId` = `emp2`.`id`
    AND
        `emp1`.`salary` > `emp2`.`salary`;

    # -------------------------
    # Method One: 显式自连接（内连接）
    SELECT
        `emp1`.`name` AS `Employee`
    FROM
        `Employee` `emp1`
    INNER JOIN
        `Employee` `emp2` ON `emp1`.`managerId` = `emp2`.`id`
    WHERE
        `emp1`.`salary` > `emp2`.`salary`;
    ```
