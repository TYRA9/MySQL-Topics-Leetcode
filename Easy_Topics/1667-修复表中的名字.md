3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: SUBSTRING, SUBSTRING
    SELECT 
        `user_id`,
        CONCAT(UCASE(SUBSTRING(`name`, 1, 1)), LCASE(SUBSTRING(`name`, 2))) AS `name`
    FROM 
        `Users`
    ORDER BY
        `user_id`;


    # ------------------------------
    # Method Two : LEFT, SUBSTRING
    SELECT
        `user_id`,
        CONCAT(UCASE(LEFT(`name`, 1)), LCASE(SUBSTRING(`name`, 2, LENGTH(`name`) - 1))) AS `name`
    FROM
        `Users`
    ORDER BY
        `user_id`;
    ```
