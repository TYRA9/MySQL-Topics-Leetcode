3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    -- Do not need self-connect because the two columns has no correlation.

    # Method One: GROUP BY
    SELECT
        `user_id`,
        COUNT(`follower_id`) AS `followers_count`
    FROM
        `Followers`
    GROUP BY
        `user_id`
    ORDER BY
        `user_id`;

    # ------------------------------
    # Method Two : Window Function (DISTINCT + OVER)
    SELECT
        DISTINCT `user_id`,
        COUNT(`follower_id`) OVER (PARTITION BY `user_id`) AS `followers_count`
    FROM
        `Followers`
    ORDER BY
        `user_id`;
    ```
