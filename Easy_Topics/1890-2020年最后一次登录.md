3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One : GROUP BY
    SELECT 
        `user_id`,
        MAX(`time_stamp`) AS `last_stamp`
    FROM
        `Logins`
    WHERE
        `time_stamp` BETWEEN '2020-01-01 00:00:00' AND '2020-12-31 23:59:59'
    GROUP BY
        `user_id`;
    ```
