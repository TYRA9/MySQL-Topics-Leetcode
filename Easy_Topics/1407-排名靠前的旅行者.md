3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: LEFT JOIN
    SELECT
        `Users`.`name`,
        IFNULL(SUM(`Rides`.`distance`), 0) AS `travelled_distance`
    FROM
        `Users`
    LEFT JOIN
        `Rides` ON `Users`.`id` = `Rides`.`user_id`
    GROUP BY
        `Rides`.`user_id`
    ORDER BY
        `travelled_distance` DESC, `Users`.`name`ASC;


    # -------------------------
    # Method Two: RIGHT JOIN
    SELECT
        `Users`.`name`,
        IFNULL(SUM(`Rides`.`distance`), 0 ) AS `travelled_distance`
    FROM
        `Rides`
    RIGHT JOIN
        `Users` ON `Rides`.`user_id` = `Users`.`id`
    GROUP BY
        `Rides`.`user_id`
    ORDER BY
        `travelled_distance` DESC, `Users`.`name` ASC;
    ```
