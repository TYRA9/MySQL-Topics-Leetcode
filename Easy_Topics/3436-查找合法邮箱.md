3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    -- In MySQL, REGEXP_LIKE returns true if any part of 
    -- the string matches the pattern and we do not tell the regex 
    -- to start looking from the very beginning of the string.
    # Method One: REGEXP_LIKE
    SELECT
        `user_id`,
        `email`
    FROM
        `Users`
    WHERE
        REGEXP_LIKE(`email`, '^[0-9a-zA-Z_]+@[a-zA-Z]+\\.com$')
    ORDER BY
        `user_id`;


    # ------------------------------
    # Method Two: REGEXP
    SELECT
        `user_id`,
        `email`
    FROM
        `Users`
    WHERE
        `email` REGEXP '^[0-9a-zA-Z_]+@[a-zA-Z]+\\.com$'
    ORDER BY
        `user_id`;
    ```
