3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: WHERE
    SELECT
        `product_name`,
        `year`,
        `price`
    FROM
        `Sales`, `Product`
    WHERE
        `Sales`.`product_id` = `Product`.`product_id`;

    # -------------------------
    # Method Two : INNER JOIN
    SELECT 
        `product_name`,
        `year`,
        `price`
    FROM
        `Sales` 
    INNER JOIN
        `Product` ON `Sales`.`product_id` = `Product`.`product_id`;
    ```
