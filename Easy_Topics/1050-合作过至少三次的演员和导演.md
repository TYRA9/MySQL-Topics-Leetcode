3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: GROUP BY
    SELECT
        `actor_id`,
        `director_id`
    FROM
        `ActorDirector`
    GROUP BY
        `actor_id`, `director_id`
    HAVING
        COUNT(`timestamp`) >= 3;

    # -------------------------
    # Method Two: Window Function
    SELECT
        DISTINCT `actor_id`,
        `director_id`
    FROM (
        SELECT
            `actor_id`,
            `director_id`,
            COUNT(`timestamp`) OVER (PARTITION BY `actor_id`, `director_id`) AS `cooperation_count`
        FROM
            `ActorDirector`
    ) AS `Subquery`
    WHERE
        `cooperation_count` >= 3;
    ```
