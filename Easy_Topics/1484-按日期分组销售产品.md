3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: GROUP_CONCAT
    SELECT
        `sell_date`,
        COUNT(DISTINCT `product`) AS `num_sold`,
        GROUP_CONCAT(DISTINCT `product`  ORDER BY `product` SEPARATOR ',') AS `products`
    FROM
        `Activities`
    GROUP BY
        `sell_date`
    ORDER BY
        `sell_date`;
    ```
