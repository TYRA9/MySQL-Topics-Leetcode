3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One : Left Join
    SELECT
        `name`,
        `bonus`
    FROM
        `Employee`
    LEFT JOIN 
        `Bonus` ON `Employee`.`empId` = `Bonus`.`empId`
    WHERE 
        `bonus` < 1000 OR `bonus` IS NULL;

    # -------------------------
    # Method Two : Right Join
    SELECT
        `name`,
        `bonus`
    FROM
        `Bonus`
    RIGHT JOIN 
        `Employee` ON `Bonus`.`empId` = `Employee`.`empId`
    WHERE 
        `bonus` < 1000 OR ISNULL(`bonus`);
    ```
