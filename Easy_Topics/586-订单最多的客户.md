3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: GROUP BY + ORDER BY 
    SELECT
        `customer_number`
    FROM
        `Orders`
    GROUP BY
        `customer_number`
    ORDER BY
        COUNT(*) DESC
    LIMIT 1;

    # -------------------------
    # Method Two: Window Function(MySQL 8.0+)
    SELECT
        `customer_number`
    FROM (
        SELECT
            `customer_number`,
            COUNT(*) AS `order_count`,
            RANK() OVER (ORDER BY COUNT(*) DESC) AS `rank`
        FROM
            `Orders`
        GROUP BY
            `customer_number`
    ) AS `RankedTable`
    WHERE
        `rank` = 1;
    ```
