3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One: GROUP BY + HAVING
    SELECT
        `class`
    FROM
        `Courses`
    GROUP BY
        `class`
    HAVING 
        COUNT(*) >= 5;

    # -------------------------
    # Method Two: Subquery
    SELECT
        `class`
    FROM (
        SELECT
            `class`,
            COUNT(student) AS `student_count`
        FROM
            `Courses`
        GROUP BY 
            `class`
    ) AS `ClassCounts`
    WHERE
        `student_count` >= 5;
    ```
