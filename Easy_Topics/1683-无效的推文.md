3. **答案**（纯代码）： 
      
    ```sql
    # Write your MySQL query statement below
    # Method One : LENGTH() func
    SELECT
        `tweet_id`
    FROM
        `Tweets`
    WHERE
        LENGTH(`content`) > 15;

    # ------------------------------
    # Method Two : CHAR_LENGTH() func
    SELECT
        `tweet_id`
    FROM
        `Tweets`
    WHERE
        CHAR_LENGTH(`content`) > 15;

    # ------------------------------
    # Method Three : check up on the 16th character
    SELECT
        `tweet_id`
    FROM
        `Tweets`
    WHERE
        SUBSTRING(`content`, 16, 1) != '';
    ```
